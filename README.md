Essential settings and customizations to improve accessibility of helix using the Xscriptor themes.


![Preview](./assets/preview/preview01.png)



## Installation

- Download themes folder.
- copy the folder on:

```bash
~/.config/helix/
```

run
```bash
#for example:
helix file.ext

#then type
:theme xscriptor
#or
:theme xscriptor_candypop
#pressEnter
```
